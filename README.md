## 🚀 Instalando `<Blueteco>`

Para instalar o `<Blueteco>`, siga estas etapas:

Clone o projeto:
```
<git clone https://github.com/danielafarias/Blueteco.git>
```
Em seguida, adicione as bibliotecas e dependências necessárias:
```
<cd Blueteco>
```
```
<cd server>
```
```
<npm init>
```

## 🧑‍💻 Usando `<Blueteco>`

Para usar `<Blueteco>`:

```
<npm run start:dev>
```
